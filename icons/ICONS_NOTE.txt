Icon files are needed for the extension:

- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

For testing, the extension will work without custom icons - Chrome will use default placeholder icons.

To create proper icons:
1. Create a simple icon design (e.g., a brain or AI symbol)
2. Export in 16x16, 48x48, and 128x128 pixel sizes
3. Save as PNG files in this directory

You can use tools like:
- Figma (https://figma.com)
- Canva (https://canva.com)
- GIMP (https://gimp.org)
- Or any online icon generator
