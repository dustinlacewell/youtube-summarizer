YouTube Summarizer Settings

Configure your Claude API key to start summarizing videos

Get your API key from console.anthropic.com
Select which Claude model to use for summarization. Opus provides highest quality, Sonnet is faster and cheaper.

How to use:

  1. Enter your Claude API key above and click "Save"
  2. Navigate to any YouTube video
  3. Click the "Summarize" button below the video
  4. View the AI-generated summary in a modal

Privacy:

Your API key is stored securely using Chrome's sync storage and is encrypted by Chrome. It syncs across your devices but is never sent to third parties.
