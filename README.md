# YouTube Summarizer

A Chrome extension that summarizes YouTube videos using Claude AI.

## Features
- Clicking the extension icon opens the transcript panel
- Adds a "Summarize" button directly to YouTube's transcript panel
- Works with any video that has transcripts/captions
- Markdown formatted summaries

## Installation

1. **Clone or download this repository**

2. **Get a Claude API key**
   - Visit [https://console.anthropic.com/](https://console.anthropic.com/)
   - Create an account or sign in
   - Generate an API key

3. **Load the extension in Chrome**
   - Open Chrome and navigate to `chrome://extensions/`
   - Enable "Developer mode" (toggle in top right)
   - Click "Load unpacked"
   - Select the extension directory

4. **Configure your API key**
   - Right-click the extension icon in Chrome toolbar
   - Click "Options"
   - Enter your Claude API key
   - Click "Save"
